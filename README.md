# Quiz Program

## Description

**Quiz Program** is a simple console-based quiz application built with C#. This application features a set of five questions related to South Africa, providing users with an engaging way to test their knowledge about the country. The quiz is designed to be easy to use and runs in the terminal, making it accessible for users of all levels.

## Features

- **Five South African-themed Questions**: Test your knowledge with questions about South African geography, history, and culture.
- **Simple Console Interface**: The quiz runs in the command line, making it lightweight and easy to use.
- **Input Validation**: Ensures that user input is valid and within the expected range.
- **Score Tracking**: Keeps track of the user’s score and displays it at the end of the quiz.

## How to Run

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Yourboy-2/Quiz-Game-Program
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd QuizProgram
   ```

3. **Open the Project in Visual Studio**:
   - Open `Quiz Program.sln` in Visual Studio.

4. **Run the Application**:
   - Press **F5** or click on the **Start** button to run the application.

## Usage

1. **Answer Questions**: The quiz will present questions one by one. Enter the number corresponding to your answer and press **Enter**.
2. **Receive Feedback**: After each question, the application will tell you if your answer was correct or incorrect.
3. **View Final Score**: At the end of the quiz, your total score will be displayed.
